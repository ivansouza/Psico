# 🌀 Psico

> *Psicodelic app — um playground de visualização, matemática, áudio e IA, tudo em HTML puro.*

**250+ experimentos** rodando direto no navegador. Sem build, sem framework, sem dependência — só abrir o arquivo e ver o que acontece.

## 🌌 O que tem aqui

O repositório é um laboratório vivo. Cada `.html` é uma ilha — autossuficiente, exploratória, muitas vezes alucinada.

## 🚀 Como usar

```bash
git clone https://github.com/ivansouza/Psico.git
cd Psico
# Abra qualquer .html no navegador
```

Ou [acesse online via GitHub Pages](https://ivansouza.github.io/Psico/) — o `index.html` é um dashboard premium com busca, filtros e grid de cards.

## 🧠 Stack

- **HTML + CSS + JS vanilla** — 100% dos experimentos
- **CDNs ocasionais**: Three.js, Tailwind, Font Awesome, Google Fonts
- **WebGPU/WebGL** para shaders e inferência local (BitNet, Whisper, TTS)
- **Zero build tools**. Zero bundlers. Zero arrependimentos.

## 🎯 Filosofia
