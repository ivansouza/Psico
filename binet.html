Fórmula de Binet

A fórmula de Binet permite encontrar o n-ésimo termo da sequência de Fibonacci de forma direta.

$$F_n = \frac{\phi^n - \psi^n}{\sqrt{5}}$$
$$\phi = \frac{1 + \sqrt{5}}{2}$$
$$\psi = \frac{1 - \sqrt{5}}{2}$$

Sequência de Fibonacci até $n$

A Lógica Matemática

Definida de forma recursiva pela relação: $$F_n = F_{n-1} + F_{n-2}$$ com os valores iniciais $F_0 = 0$ e $F_1 = 1$.

A fórmula de Binet fornece uma solução analítica para esta recorrência. Ela utiliza a Proporção Áurea ($\phi$), que é a raiz positiva da equação $x^2 - x - 1 = 0$: $$\phi = \frac{1 + \sqrt{5}}{2} \approx 1,6180339887...$$

Como o valor absoluto da base conjugada $|\psi| = |(1 - \sqrt{5})/2| \approx 0,618$ é inferior a $1$, o termo $\psi^n$ converge rapidamente para zero quando $n$ aumenta: $$\lim_{n \to \infty} \psi^n = 0$$

Isto significa que, para qualquer $n \ge 0$, o n-ésimo número de Fibonacci pode ser encontrado simplesmente arredondando o seguinte cálculo para o inteiro mais próximo: $$F_n = \text{round}\left( \frac{\phi^n}{\sqrt{5}} \right)$$

Nota Computacional: Em JavaScript, a precisão total para inteiros é mantida até $n = 78$. Para valores superiores, a calculadora utiliza notação científica de alta precisão para evitar erros de arredondamento visual.
